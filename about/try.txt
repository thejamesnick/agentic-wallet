## The Security Model: "Cold Storage, Hot Execution"

```
┌─────────────────────────────────────────┐
│  AT REST (Encrypted)                    │
│  - Private keys encrypted with AES-256  │
│  - Password/SSH-key protected           │
│  - Stored in ~/.paw/wallets/            │
├─────────────────────────────────────────┤
│  IN MEMORY (Decrypted)                  │
│  - Only decrypted during signing        │
│  - Zeroed after use                     │
│  - Never logged or persisted            │
├─────────────────────────────────────────┤
│  EXECUTION (Sign & Broadcast)           │
│  - Transaction signed in-memory         │
│  - Immediate broadcast to Solana        │
│  - Key cleared from memory              │
└─────────────────────────────────────────┘
```

## Implementation: SSH-Style Encryption

### 1. **Key Generation & Encryption**

```typescript
// src/core/crypto/vault.ts
import { generateKeyPair, box, randomBytes } from 'tweetnacl';
import { pbkdf2Sync, createCipheriv, createDecipheriv, scryptSync } from 'crypto';
import { readFileSync, writeFileSync, existsSync, mkdirSync } from 'fs';
import { homedir } from 'os';
import { join } from 'path';
import { execSync } from 'child_process';

export class SecureVault {
  private vaultPath: string;
  private masterKey?: Buffer; // Only held in memory when unlocked
  
  constructor() {
    this.vaultPath = join(homedir(), '.paw', 'vault');
    this.ensureVaultExists();
  }

  // Generate new agent wallet with SSH-style encryption
  async createAgentWallet(agentId: string, authMethod: AuthMethod): Promise<WalletCredentials> {
    // Generate Solana keypair
    const keypair = Keypair.generate();
    const secretKey = keypair.secretKey; // 64 bytes
    
    // Derive encryption key using SSH-style method
    const encryptionKey = await this.deriveKey(authMethod);
    
    // Encrypt private key
    const encrypted = this.encrypt(secretKey, encryptionKey);
    
    // Store metadata
    const walletData: EncryptedWallet = {
      agentId,
      publicKey: keypair.publicKey.toBase58(),
      encryptedKey: encrypted.toString('base64'),
      authType: authMethod.type,
      createdAt: Date.now(),
      lastUsed: null,
    };
    
    this.saveWallet(walletData);
    
    // Immediately clear sensitive data
    secretKey.fill(0);
    
    return {
      agentId,
      publicKey: keypair.publicKey.toBase58(),
      address: keypair.publicKey.toBase58(),
    };
  }

  // SSH-style key derivation
  private async deriveKey(auth: AuthMethod): Promise<Buffer> {
    switch (auth.type) {
      case 'password':
        // PBKDF2 like SSH keys
        return pbkdf2Sync(
          auth.password,
          auth.salt || randomBytes(16),
          100000, // 100k iterations
          32,
          'sha256'
        );
        
      case 'ssh-agent':
        // Use system SSH agent for decryption
        return this.deriveFromSSHAgent(auth.keyId);
        
      case 'hardware':
        // YubiKey/TPM integration
        return this.deriveFromHardware(auth.slot);
        
      case 'system-keychain':
        // macOS Keychain / Windows DPAPI / Linux keyring
        return this.deriveFromKeychain(auth.service);
    }
  }

  // Real-time decryption for signing only
  async signTransaction(
    agentId: string,
    transaction: Transaction,
    auth: AuthMethod
  ): Promise<string> {
    // 1. Load encrypted wallet
    const wallet = this.loadWallet(agentId);
    
    // 2. Derive decryption key (real-time)
    const decryptionKey = await this.deriveKey(auth);
    
    try {
      // 3. Decrypt private key (in memory only)
      const secretKey = this.decrypt(
        Buffer.from(wallet.encryptedKey, 'base64'),
        decryptionKey
      );
      
      // 4. Create keypair and sign
      const keypair = Keypair.fromSecretKey(secretKey);
      transaction.sign(keypair);
      
      // 5. Send transaction
      const signature = await this.connection.sendRawTransaction(
        transaction.serialize()
      );
      
      return signature;
      
    } finally {
      // 6. CRITICAL: Zero out memory
      if (decryptionKey) decryptionKey.fill(0);
      // secretKey is scoped to try block, will be GC'd
    }
  }
}
```

### 2. **SSH-Agent Integration (The Cool Part)**

```typescript
// src/core/crypto/ssh-bridge.ts
import { execSync, spawn } from 'child_process';
import { createDecipheriv, randomBytes, createCipheriv } from 'crypto';

export class SSHAgentBridge {
  private agentSocket: string;

  constructor() {
    // Get SSH agent socket from environment
    this.agentSocket = process.env.SSH_AUTH_SOCK || '';
  }

  // Check if SSH agent is running
  isAvailable(): boolean {
    try {
      execSync('ssh-add -l', { stdio: 'pipe' });
      return true;
    } catch {
      return false;
    }
  }

  // Use SSH key to encrypt/decrypt wallet key
  async encryptWithSSHKey(walletKey: Buffer, keyId: string): Promise<EncryptedPayload> {
    // Generate ephemeral AES key
    const aesKey = randomBytes(32);
    const iv = randomBytes(16);
    
    // Encrypt wallet key with AES
    const cipher = createCipheriv('aes-256-gcm', aesKey, iv);
    const encryptedWallet = Buffer.concat([cipher.update(walletKey), cipher.final()]);
    const authTag = cipher.getAuthTag();
    
    // Encrypt AES key with SSH public key using openssl
    const encryptedAesKey = this.rsaEncryptWithSSH(aesKey, keyId);
    
    // Clear AES key from memory
    aesKey.fill(0);
    
    return {
      encryptedKey: encryptedWallet.toString('base64'),
      encryptedAesKey: encryptedAesKey.toString('base64'),
      iv: iv.toString('base64'),
      authTag: authTag.toString('base64'),
      keyId,
    };
  }

  // Decrypt using SSH agent (real-time only)
  async decryptWithSSHAgent(payload: EncryptedPayload): Promise<Buffer> {
    // Decrypt AES key using SSH agent
    const aesKey = this.rsaDecryptWithSSHAgent(
      Buffer.from(payload.encryptedAesKey, 'base64'),
      payload.keyId
    );
    
    try {
      // Decrypt wallet key
      const decipher = createDecipheriv(
        'aes-256-gcm',
        aesKey,
        Buffer.from(payload.iv, 'base64')
      );
      decipher.setAuthTag(Buffer.from(payload.authTag, 'base64'));
      
      const decrypted = Buffer.concat([
        decipher.update(Buffer.from(payload.encryptedKey, 'base64')),
        decipher.final()
      ]);
      
      return decrypted;
      
    } finally {
      // CRITICAL: Clear AES key immediately
      aesKey.fill(0);
    }
  }

  private rsaEncryptWithSSH(data: Buffer, keyId: string): Buffer {
    // Use openssl to encrypt with SSH public key
    const result = execSync(
      `openssl pkeyutl -encrypt -in - -pubin -inkey <(ssh-keygen -f "${keyId}" -e -m PKCS8)`,
      { input: data, shell: '/bin/bash' }
    );
    return result;
  }

  private rsaDecryptWithSSHAgent(data: Buffer, keyId: string): Buffer {
    // Use SSH agent for decryption (requires unlocked key)
    // This only works if SSH agent has the key loaded and unlocked
    const result = execSync(
      `openssl pkeyutl -decrypt -in - -inkey <(ssh-add -L | grep "${keyId}")`,
      { input: data, shell: '/bin/bash' }
    );
    return result;
  }
}
```

### 3. **Session-Based "Unlock" System**

```typescript
// src/core/session/manager.ts
import { EventEmitter } from 'events';

export class SessionManager extends EventEmitter {
  private sessions: Map<string, Session> = new Map();
  private masterKey?: Buffer; // Only stored when explicitly unlocked
  
  // Unlock wallet for a session (like sudo timeout)
  async unlock(agentId: string, auth: AuthMethod, duration: number = 3600): Promise<Session> {
    const vault = new SecureVault();
    
    // Verify auth and get master key
    this.masterKey = await vault.deriveKey(auth);
    
    // Create session with auto-lock timer
    const session: Session = {
      agentId,
      unlockedAt: Date.now(),
      expiresAt: Date.now() + (duration * 1000),
      masterKey: this.masterKey, // Reference only, cleared on lock
      signCount: 0,
    };
    
    this.sessions.set(agentId, session);
    
    // Auto-lock timer
    setTimeout(() => this.lock(agentId), duration * 1000);
    
    this.emit('unlocked', { agentId, expiresAt: session.expiresAt });
    
    return session;
  }

  // Sign within unlocked session (no password needed)
  async signWithSession(agentId: string, transaction: Transaction): Promise<string> {
    const session = this.sessions.get(agentId);
    
    if (!session || Date.now() > session.expiresAt) {
      throw new Error('Session expired or not unlocked. Run: paw unlock --agent-id ' + agentId);
    }
    
    const vault = new SecureVault();
    
    // Decrypt, sign, and immediately re-encrypt
    const signature = await vault.signTransactionWithKey(
      agentId,
      transaction,
      session.masterKey // Use cached key
    );
    
    session.signCount++;
    session.lastUsed = Date.now();
    
    return signature;
  }

  // Explicit lock (like screen lock)
  lock(agentId: string): void {
    const session = this.sessions.get(agentId);
    if (session) {
      // Zero out master key reference
      if (this.masterKey) {
        this.masterKey.fill(0);
        this.masterKey = undefined;
      }
      
      this.sessions.delete(agentId);
      this.emit('locked', { agentId });
    }
  }

  // Lock all sessions (system sleep, etc.)
  lockAll(): void {
    for (const agentId of this.sessions.keys()) {
      this.lock(agentId);
    }
  }
}
```

## CLI Commands for Security

```bash
# Initialize with SSH-key protection
paw init --agent-id trading-bot --auth ssh --ssh-key ~/.ssh/id_ed25519

# Unlock for session (like sudo)
paw unlock --agent-id trading-bot --duration 3600  # 1 hour
# Enter SSH key passphrase (if not in agent)

# Now agent can sign without password
paw swap --agent-id trading-bot --from SOL --to USDC --amount 1
# Auto-signed using session key

# Or stay locked - require password per transaction
paw swap --agent-id trading-bot --from SOL --to USDC --amount 1 --auth password

# Check session status
paw status --agent-id trading-bot
# Output: Unlocked (expires in 45:23) | 12 signatures | Last: 2 min ago

# Manual lock
paw lock --agent-id trading-bot

# Auto-lock on system sleep
paw config --auto-lock-on-sleep true
```

## The "Agentic" Security Flow

```
┌─────────────────────────────────────────┐
│  AGENT WANTS TO TRADE                   │
│  "Swap 1 SOL to USDC"                   │
└──────────────┬──────────────────────────┘
               │
               ▼
┌─────────────────────────────────────────┐
│  PAW CHECKS: Is session unlocked?       │
│  ├─ YES → Sign immediately              │
│  └─ NO  → Require auth                  │
│      ├─ SSH key (if available)          │
│      ├─ Password (fallback)             │
│      └─ Hardware key (YubiKey)          │
└──────────────┬──────────────────────────┘
               │
               ▼
┌─────────────────────────────────────────┐
│  REAL-TIME DECRYPTION                   │
│  1. Load encrypted key from disk        │
│  2. Decrypt in memory (never to disk)   │
│  3. Sign transaction                    │
│  4. Zero memory buffer                  │
│  5. Return signature                    │
└──────────────┬──────────────────────────┘
               │
               ▼
┌─────────────────────────────────────────┐
│  BROADCAST TO SOLANA                    │
│  - Transaction sent                     │
│  - Key no longer in memory              │
│  - Session continues (if unlocked)      │
└─────────────────────────────────────────┘
```

## Why This Wins on Security

| Threat | Mitigation |
|--------|------------|
| **Key theft from disk** | Always encrypted, AES-256-GCM |
| **Memory dump** | Keys only in memory during signing, zeroed after |
| **Malware keylogging** | SSH-agent integration, hardware key support |
| **Agent compromise** | Session limits, auto-lock, spending caps |
| **Supply chain attack** | Signed binaries, reproducible builds |
| **Side-channel** | Constant-time crypto operations |
