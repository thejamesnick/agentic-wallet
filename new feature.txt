PAW - New Features List

1. Address with QR Code ✅
   - Display wallet address as a scannable QR code in terminal
   - Makes it easy to receive funds from mobile wallets
   - Command: paw address <agent-id> --qr
   - Useful for: Quick funding, mobile wallet transfers

2. Export Private Key ✅
   - Export wallet private key for backup or migration
   - Command: paw export <agent-id>
   - Security: 
     * Shows warning about security risks
     * Interactive: Requires typing agent ID to confirm
     * Automated: Use --confirm flag to skip prompt
     * Example: paw export agent-alice --confirm agent-alice
   - Useful for: Wallet backup, importing to Phantom/Solflare/other wallets
   - AI Agent friendly: Can automate with --confirm flag
   - Note: Private key can be imported into any Solana wallet

3. Import Wallet ✅
   - Import existing wallet from private key
   - Allows using existing wallets with PAW
   - Command: paw import <agent-id> --private-key "base58key"
   - Useful for: Migrating existing wallets, using Phantom/Solflare wallets with PAW
   - Note: Only private key import supported (not seed phrases)

4. Multi-Send ✅
   - Send SOL to multiple addresses in one command
   - Saves time and transaction fees (all in single transaction)
   - Command: paw multi-send <agent-id> --addresses addr1,addr2,addr3 --amounts 0.1,0.2,0.3
   - Example: paw multi-send agent-alice --addresses Bob,Charlie --amounts 0.5,0.5
   - Useful for: Airdrops, batch payments, team payouts
   - Note: Currently supports SOL only (not SPL tokens)

Status: ✅ All features completed!
Priority: High (all 4 features are essential)

Future Enhancements:
- Add transaction size limit check for multi-send (max ~15-20 recipients)
- Support SPL token multi-send (not just SOL)
- CSV file import for multi-send recipients
