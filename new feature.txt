PAW - New Features List

1. Address with QR Code ✅
   - Display wallet address as a scannable QR code in terminal
   - Makes it easy to receive funds from mobile wallets
   - Command: paw address <agent-id> --qr
   - Useful for: Quick funding, mobile wallet transfers

2. Export Seed Phrase & Private Key 
   - Export wallet seed phrase (12/24 words) for backup
   - Export raw private key for importing to other wallets
   - Command: paw export <agent-id> --seed or --private-key
   - Security: 
     * Shows warning about security risks
     * Interactive: Requires typing agent ID to confirm
     * Automated: Use --confirm flag to skip prompt
     * Example: paw export agent-alice --seed --confirm agent-alice
   - Useful for: Wallet backup, migration to other wallets
   - AI Agent friendly: Can automate with --confirm flag

3. Import Wallet
   - Import existing wallet from seed phrase or private key
   - Allows using existing wallets with PAW
   - Command: paw import <agent-id> --seed "word1 word2..." or --private-key "base58key"
   - Useful for: Migrating existing wallets, using hardware wallet addresses

4. Multi-Send
   - Send SOL/tokens to multiple addresses in one command
   - Saves time and transaction fees
   - Command: paw multi-send <agent-id> --addresses addr1,addr2,addr3 --amounts 0.1,0.2,0.3
   - Or use CSV file: paw multi-send <agent-id> --file recipients.csv
   - Useful for: Airdrops, batch payments, team payouts

Status: Planning phase
Priority: High (all 4 features are essential)
